# 🐍 Snake Game

A modern Snake Game built using React and HTML Canvas with smooth gameplay and a polished UI.

---

## 🚀 Features

- 🎮 Smooth snake movement
- 🍬 Random colored food
- 📈 Score tracking
- 💀 Collision detection (walls & self)
- 🔄 Restart functionality
- 🌗 Dark / Light theme toggle
- ✨ Clean and responsive UI

---

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- HTML Canvas
- CSS3
